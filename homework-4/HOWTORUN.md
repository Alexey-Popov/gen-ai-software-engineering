# How to Run

## Prerequisites
- **Node.js >= 20** and **npm** — for the sample app and tests.
- **Claude Code CLI** — for the pipeline runner. Install: <https://docs.claude.com/claude-code>
- **Anthropic API access** — either `ANTHROPIC_API_KEY` env var or `claude /login` with a Pro/Team subscription.

## Install dependencies
```bash
cd homework-4
npm install
```

## Run the sample application
```bash
npm start
# server: http://localhost:3000
# try:    curl 'http://localhost:3000/notes?page=1&perPage=2'
```

## Run tests (baseline)
```bash
npm test
```
Before the pipeline runs, expect **4 failed, 1 passed** — these are the seeded bugs (pagination, search-crash).
See `docs/screenshots/01-baseline-failures.png`.

## Run the 6-agent pipeline (single command)
```bash
npm run pipeline
# or equivalently
./run-pipeline.sh
# different bug bundle:
BUG_ID=002 ./run-pipeline.sh
```

What it does, step by step:
1. Sanity-checks `bug-context.md`, `claude` CLI, and `node_modules/`.
2. Syncs `agents/*.agent.md` → `.claude/agents/*.md` and `skills/*.md` → `.claude/skills/` so Claude Code can discover them.
3. Starts **one orchestrator Claude session** that uses the Agent tool to dispatch all 6 subagents in order:
   1. `bug-researcher` → `context/bugs/001/research/codebase-research.md`
   2. `research-verifier` → `verified-research.md` (halts if `Pipeline action: STOP`)
   3. `bug-planner` → `implementation-plan.md` (halts if `BLOCKED`)
   4. `bug-fixer` → `fix-summary.md` + edits in `src/` (halts if `failed`)
   5. `security-verifier` → `security-report.md` (does not halt)
   6. `unit-test-generator` → `test-report.md` + new `tests/*.test.js`

## Verify after the pipeline
```bash
npm test
```
Expect all tests green (the 4 previously-failing ones + new ones generated by the Unit Test Generator).

## Reset for a fresh run
Reverts the seeded bugs and clears the agent artifacts:
```bash
git restore src/ tests/
rm -rf context/bugs/001/research \
       context/bugs/001/implementation-plan.md \
       context/bugs/001/fix-summary.md \
       context/bugs/001/security-report.md \
       context/bugs/001/test-report.md
```

## Troubleshooting
- **`'claude' CLI not found`** — install Claude Code, ensure it's on PATH (`which claude`).
- **Permission denied for `./run-pipeline.sh`** — `chmod +x run-pipeline.sh`.
- **API key errors** — `claude /login` or `export ANTHROPIC_API_KEY=…`.
- **Bug Fixer can't find `npm test`** — run `npm install` first; the orchestrator does not install deps.
